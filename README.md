- [Node.js](https://nodejs.org/) v14 or higher

### ğŸ‘¾ Installation

1. **Clone the repository:**

   ```sh
   git clone https://github.com/playwora/wora.git
   cd wora
   ```

2. **Install the dependencies:**

   ```sh
   bun install
   ```

3. **Start the application:**

   ```sh
   bun run dev
   ```

4. **Build the application**

   ```sh
   bun run build
   ```
